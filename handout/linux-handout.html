Linux Commands Quick Reference

Navigation

Print working directory Show your current location in the directory structure


pwd

List List the files and folders in a directory


ls

Change directory Switch to a different directory


cd foldername

Special Locations

  • . Here (current directory)
  • ../ One level up from the current directory
  • ~/ Home (current user's home directory)
  • / Root (top-most directory)

Accessing other machines

Secure Shell Access another machine remotely, via an encrypted, access-controlled connection

ssh [username]@[location]


ssh username@remotesite.org

Secure Copy Transfer files between 2 machines

Transfer from local to remote
scp [source path] [username]@[location]:[destination path]


scp filename.txt username@remotesite.org:~/

Transfer from remote to local
scp [username]@[location]:[source path] [destination path] 


scp username@remotesite.org:~/filename.txt .

Creating/Managing Files & Folders

Make directory Create a new (empty) folder


mkdir foldername

Touch Create a new (empty) file


touch newfile.txt

Copy Copy files/folders

cp [source] [destination]

Copy a file or an empty folder


cp file1.txt file2.txt

Copy a folder and its contents


cp -r folder1 folder2

Move Move or rename files/folders

mv [source] [destination]

To move a file/folder, specify a path and a name


mv file1.txt folder1/file1.txt
mv folder1 documents/folder1

To rename a file/folder, specify only a new name


mv file1.txt file2.txt
mv folder1 folder2

Remove Delete files/folders

Delete a file or an empty folder


rm filename.txt

Delete a folder and its contents


rm -r foldername

Searching

Find Search for files/folders

find [location] [options] [search terms]


find ~/ "filename"

Grep Search for text within files

grep [options] [search terms] [file(s)]

Search for text within a single file


grep searchterm filename.txt

Search for text in all files within a folder


grep -r searchterm foldername

Permissions

Permission types:

  • r (read) view a file/folder's content
  • w (write) edit a file/folder
  • x (execute) run a file (if it's a program/script) OR enter a folder

User roles:

  • u (user/owner) owner of the file/folder
  • g (group) group that a file/folder belongs to
  • o (others) everyone who is not the owner or a group member
  • a (all) all roles (user, group AND others)

Chmod Change permissions for files/folders

chmod [user role][+ add or - remove][permission type] [file]

Change permission for a single file


chmod g+w filename.txt

Change permission for all files in a folder


chmod g-w foldername/*

Sudo Run any single command as 'root' user


sudo chmod g+w newproject/*

Editing Files

Open Open files in an external application

Open a file in its default application


open filename.txt

Open a file in a specific application


open filename.txt -a textedit

Vi Open or create a file in the Vi command line text editor

Open an existing file in Vi


vi filename.txt

Create a new file and open it in Vi
(Note: the new file won't be saved until you save it in using :w or :wq)


vi newfile.txt

Vi Commands These only work when you're inside the Vi editor!

  • i Insert mode (add/edit text)
  • esc Command mode (search/navigate around the file without affecting text)
  • :w Save without quitting (must be in command mode)
  • :wq Save and quit Vi (must be in command mode)
  • :q Quit without saving (must be in command mode)
  • :q! Quit without saving, even if Vi doesn't want to (must be in command mode)

Concatenate View a file's contents OR combine multiple files into one

View the contents of a file


cat filename.txt

Combine multiple files into one: cat [file1] [file2] > [newfile]


cat file1.txt file2.txt > file3.txt
